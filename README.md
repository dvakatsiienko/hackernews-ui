# Hackernews UI

Built with the following technology stack:

-   React (Next.js)
-   TypeScript
-   Apollo Client
-   Apollo Server
-   SQLite DB
-   Prisma ORM
-   SQLite DB
-   Vercel cloud deployment for UI
-   Railway cloud deployment for API

The **backend** (**API**) for this project is [located here](https://github.com/dvakatsiienko/hackernews-api).
